Boot Loader Help

Benvenuti in DSS LIVE

Tramite questo menu potete selezionare la funzione desiderata. Se doveste incontrare delle difficolt  navigando attraverso questo sistema di aiuto, premete F1 per avere la descrizione del sistema di aiuto. Le funzioni principali del menu sono:

Default: Questa modalit  di installazione funziona sulla maggior parte delle macchine. Se il sistema si blocca durante il processo di boot o si verificano delle difficolt  in relazione al rilevamento delle vostre componenti hardware, come ad esempio controller di dischi o schede di rete, provate con una delle seguenti opzioni di installazione.

DSSLIVE -- ACPI disabilitato: Molti dei computer attualmente in commercio hanno delle implementazioni ACPI incomplete o errate. Questa selezione disabilita il supporto ACPI nel kernel, ma rimangono comunque abilitate tante caratteristiche interessanti da un punto di vista della performance, come ad esempio DMA per dischi rigidi IDE.

Verifica della memoria: Questa verifica utile non solo per controllare l'installazione di nuovi moduli di memoria. Si tratta di una dura prova per la maggior parte del vostro sistema e potrebbe rilevare dei problemi dovuti all'hardware.

Opzioni di boot: Le opzioni di boot possono cambiare completamente il comportamento del vostro sistema. Si tratta di settaggi per il kernel.

F1 Aiuto: Si tratta di un tasto che risponde in base al contesto in cui viene utilizzato. Mostrer  differenti schermate in base all'elemento attivo della schermata di avvio. Vi inoltre una descrizione del sistema di aiuto.

F2 Modo video: Qui potete scegliere tra differenti risoluzioni dello schermo durante il processo di boot. Se incontrate delle difficolt  con il boot grafico, il modo testo potrebbe essere un modo per aggirare le difficolt .

F3 Lingua: Impostate la lingua e la mappatura della tastiera utilizzate dal boot loader.

Utilizzare il sistema di aiuto

L'aiuto in linea del boot loader dipende dal contesto; fornisce delle informazioni sulla voce di menu selezionata, se state modificando delle opzioni di boot, cercher  di reperire dell'informazioni relative all' opzione su cui posizionato il cursore.

Selezione modo video

Premendo F2 ottenete un elenco dei modi video supportati dalla vostra scheda grafica. E' preselezionato il modo pi elevato supportato dal vostro schermo.

A volte succede che il monitor non venga rilevato automaticamente. In questi casi, selezionate il vostro modo preferito manualmente.

Se il vostro sistema incontra delle difficolt  con la scheda grafica durante il processo di boot, il modo testo potrebbe essere un modo utile per aggirare il problema.

DSS LIVE -- ACPI disabilitata

Hardware molto recente richiede a volte l'ACPI per amministrare gli interrupt. L'ACPI prende il posto del pi datato sistema APM.

Selezionate Installazione -- ACPI disabilitata se doveste incontrare delle difficolt durante il processo di boot del kernel. Le difficolt  risapute con macchine che non si sposano bene con ACPI sono:

* blocco del kernel in fase di boot
* le schede PCI non vengono rilevate o non vengono inzializzate correttamente

Verifica della memoria

La Verifica della memoria d l'opportunit  di mettere a dura prova l'hardware del sistema. Scopo principale della verifica quello di rilevare disfunzioni della RAM, ma sottopone a dura prova anche altre componenti del sistema.
Se non vengono rilevati degli errori nella memoria questo non una garanzia per il fatto che tutto sia a posto, comunque verranno rilevati la maggior parte degli errori.

Opzioni di boot

Sono disponibili due tipi di opzioni. Le prime interessano l'installer. Le seconde sono opzioni che interessano il kernel. Tra le opzioni pi comuni vi sono:

splash -- influisce sul comportamento dello splash screen
acpi -- advanced configuration and power interface
hostname -- usa un hostanme diverso da quello di default
username --  usa un username diverso da quello di default
password -- associa una  password al' utente
toram -- copia il contenuto del cd in ram
xmodule -- specifica il driver da usare per il server X
xconf -- specifica il frontend da usare per riconfigurare il server X
lconf --  specifica il frontend da usare per riconfigurare i locales

Opzioni per il kernel: splash

Lo splash screen è la grafica visualizzata durante l'avvio del sistema.


splash=verbose

Attiva lo splash screen, e mostra messaggi del kernel e di boot.


Ritorna alle Opzioni di boot

Opzioni per il kernel: acpi

ACPI (Advanced Configuration and Power Interface) uno standard di interfaccia posta tra sistema operativo e BIOS. Di default, acpi attivata se stato rilevato un BIOS del dopo 2000. Vi sono una serie d parametri per intervenire sul comportamento dell' ACPI:

noacpi -- non utilizzare ACPI per gestire gli interrupt PCI

Soprattutto su computer recenti, ACPI si sostituisce al vecchio sistema apm

Ritorna alle Opzioni di boot

Opzioni per il boot: hostname

Modifica l' hostname [default : dsslive]

hostname=mybox


Return to Boot Options.

Opzioni per il boot: username

Modifica lo username [default : debaser]

username=myname

Opzioni per il boot: password

Associa una password all'utente [default : no password]

password=mypass


Return to Boot Options.

Opzioni per il boot: toram

Copia l'intero cd in RAM ed effettua il boot

toram[=custom] -- custom permette scegliere queli moduli copiare


Return to Boot Options.
